# README

## Requirements

* Enable TRAE wake-up via CLI
    1. Open TRAE, press `Ctrl + Shift + P` (on Windows/Linux) or `Cmd + Shift + P` (on macOS).
    2. Type `Shell Command: Install 'trae' command in PATH` and hit enter.
    3. Follow the prompts to complete the installation.
    4. Now you can use the `trae` command in your terminal to wake up TRAE.
* Disable Word Wrap
    1. Open `Preference > Settings` via `Ctrl + ,` or `Cmd + ,` (on macOS).
    2. In `General > Editor Settings > Go to Settings`, type `Word Wrap` to find the option.
    3. Set the option to `false`.
